# Todo List API

> A RESTful Todo List API built with .NET 10.

This project follows the Todo List API specification from roadmap.sh and serves as a backend-focused exercise to improve development skills while building a portfolio with clear, practical examples.

## Overview

The API provides basic task management: creating, listing, updating, and deleting todo items. The architecture emphasizes clarity, separation of concerns, and adherence to standard REST patterns.

## Technology Stack

* C# (.NET 10)
* Entity Framework Core
* SQLite
* Minimal API structure
* JSON-based communication

## Features

* CRUD operations for todo items
* Persistent storage using SQLite
* Clean separation between domain, data, and application layers
* Simple and predictable endpoints
* Standards-based API behavior

## Setup

```bash
dotnet restore
```

## Run

```bash
dotnet run
```

## Build

```bash
dotnet build
```
